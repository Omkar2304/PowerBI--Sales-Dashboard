<h2>📌 Project Overview</h2>
<p>
This project showcases an <b>interactive Power BI Sales Dashboard</b> designed to analyze and visualize sales data.
It helps businesses track performance, identify trends, and make data-driven decisions.
</p>

<ul>
<li>Track overall sales performance</li>
<li>Identify trends and patterns</li>
<li>Monitor key business metrics</li>
<li>Enable better decision-making</li>
</ul>

<hr>

<h2>🎯 Objectives</h2>
<ul>
<li>Analyze sales across regions and categories</li>
<li>Identify top-performing products</li>
<li>Track revenue trends over time</li>
<li>Generate actionable insights</li>
</ul>

<hr>

<h2>🛠️ Tools & Technologies</h2>
<ul>
<li>Microsoft Power BI</li>
<li>Power Query (ETL)</li>
<li>DAX (Data Analysis Expressions)</li>
<li>Microsoft Excel</li>
</ul>

<hr>

<h2>🔄 Data Workflow</h2>

<h3>1️⃣ Data Extraction</h3>
<ul>
<li>Imported raw sales dataset</li>
<li>Combined multiple sources</li>
</ul>

<h3>2️⃣ Data Transformation</h3>
<ul>
<li>Removed duplicates and null values</li>
<li>Standardized formats</li>
<li>Created calculated columns</li>
</ul>

<h3>3️⃣ Data Modeling</h3>
<ul>
<li>Established table relationships</li>
<li>Used star schema</li>
<li>Created measures using DAX</li>
</ul>

<hr>

<h2>🧠 Key DAX Measures</h2>
<ul>
<li>Total Sales</li>
<li>Total Profit</li>
<li>Average Revenue</li>
<li>Sales Growth</li>
<li>Monthly Trends</li>
</ul>
